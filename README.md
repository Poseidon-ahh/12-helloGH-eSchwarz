# 12-helloGH-eSchwarz

## Beschreibung

Dieses Projekt ist unser erstes Hello World und dient dazu, den grundlegenden GitHub Workflow kennenzulernen. Der Fokus liegt auf dem Verständnis von Versionskontrolle mit Git.

## Gelernt

Repository erstellen  
Dateien committen  
Branches erstellen und wechseln  
Pull Requests öffnen  
Änderungen mergen

## Workflow

1. Repository klonen
2. Neuen Branch erstellen (feature-branch)
3. Änderungen machen und committen
4. Branch auf GitHub pushen
5. Pull Request erstellen
6. Review und Merge in main

## Ziel

Ziel ist es, ein grundlegendes Verständnis für den Standard Git Workflow zu entwickeln und erste Erfahrungen mit Zusammenarbeit zu sammeln.

Dieses Projekt dient nur zu Lernzwecken.
